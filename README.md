# Interview Questions

A collection of interview questions and their solutions

## Requirements

### Python

Activate a python 3 virtual environment
```
python3 -m pip install --user virtualenv
python3 -m virtualenv env
source env/bin/activate
```

Deactivate
```
deactivate
```
